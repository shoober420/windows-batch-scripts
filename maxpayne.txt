AMD ZEN2 CPU JPEG ERRORS FIX
 - go to Max Payne directory and backup rlmfc.dll
 - open hex editor (HxD) and edit rlmfc.dll
 - go to offset 0x256ED (in common editors, row 256E0, column 0D)
 - replace 83 E0 0F to 90 90 90
