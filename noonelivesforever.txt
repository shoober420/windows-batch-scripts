No One Lives Forever
 - download (https://www.myabandonware.com/game/the-operative-no-one-lives-forever-7q4)
 - convert bin files to iso
 - mount and install
 - download nolf modernizer patch (https://haekb.itch.io/nolf-modernizer)
 - install patch files into "C:\Program Files\Fox\No One Lives Forever"
 - launch NOLF.EXE and select "Advanced", then "Customize"
 - add MODERIZER.REZ file and enable "Always load these rez files"
